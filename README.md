# ClashOfClans-Wrapper
## Made by ProtDos | CodingLive#0001

## Installation
```bash
git clone https://github.com/ProtDos/ClashOfClans-Wrapper/
cd ClashOfClans-Wrapper
pip install -r requirements.txt
python3 main.py
```

## Usage
1. Download the files
2. Install the requirements
3. Enter your tokens in `main.py` in line 2
4. run: `python3 main.py`
5. Enjoy

## Requirements
- Python 3.6+
- pip
- requirements.txt installed
- clash of clans api token

## Features
- Player
- League
- Location
- Clans
